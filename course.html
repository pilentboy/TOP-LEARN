جنگو برنامه نویسی وب پایتون دوره متوسط رایگان

اگر با تکنولوژی های وب کار کرده باشید احتمالا با واژه ی WEB API آشنایی دارید

وب سایت هایی که با فریم ورک های برنامه نویسی وب پیاده سازی میشوند برای سرویس دهی به نرم افزار های موبایل، وب و سایر پلت فرم های دیگر نیاز به ابزاری دارند که این امکان را برای آنها فراهم کند

در این دوره ی آموزشی قصد داریم به بررسی فریم ورک Django Rest Framework بپردازیم که یک تکنولوژی منحصر بفرد برای پیاده سازی WEB API بوده و برای فریم ورک جنگو پیاده سازی شده است
